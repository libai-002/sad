您好！在 VS Code 的 Database Client 插件里执行 SQL 非常简单。我来一步步教您：

### 第一步：打开 SQL 窗口
1.  **找到侧边栏**：点击 VS Code 左侧活动栏里的 **Database** 图标（通常是一个圆柱体图标）。
2.  **找到连接**：在展开的列表里，找到您已经连接成功的那个 **127.0.0.1** 连接。
3.  **打开查询**：**右键点击**那个连接（或者连接下的具体数据库），选择 **"New Query"**（新建查询）或者 **"Open Query Console"**。
    *   此时，右侧编辑区会打开一个新的、空白的 `.sql` 文件窗口。

### 第二步：输入并运行 SQL
1.  **复制粘贴**：把下面这段代码完整复制到那个新打开的窗口里：
    ```sql
    CREATE USER 'funzone'@'%' IDENTIFIED BY 'funzone123';
    GRANT ALL PRIVILEGES ON *.* TO 'funzone'@'%';
    FLUSH PRIVILEGES;
    ```
2.  **运行**：点击窗口上方的 **"Run"** 按钮（通常是一个绿色的播放键 ▶️），或者直接按快捷键（通常是 `Ctrl + Enter` 或 `Cmd + Enter`）。

### 第三步：告诉我结果
*   如果下方输出栏显示绿色的 **"Success"** 或者 **"Query OK"**，那就说明成功了！
